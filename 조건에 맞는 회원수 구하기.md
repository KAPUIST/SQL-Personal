## 문제 설명

USER_INFO 테이블에서 2021년에 가입한 회원 중 나이가 20세 이상 29세 이하인 회원이 몇 명인지 출력하는 SQL문을 작성해주세요.

## 데이터

```
USER_ID	GENDER	AGE	JOINED
1	1	26	2021-10-05
2	0	NULL	2021-11-25
3	1	22	2021-11-30
4	0	31	2021-12-03
5	0	28	2021-12-16
6	1	24	2022-01-03
7	1	NULL	2022-01-09
```

## 문제 풀이

```
SELECT COUNT(USER_ID) as USERS
FROM USER_INFO
WHERE (AGE BETWEEN 20 AND 29) AND DATE_FORMAT(JOINED,'%Y') = '2021'

```
